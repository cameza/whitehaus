Setup steps (one time, ~10 min)
  1. Go to sheets.google.com → create a new spreadsheet named Whitehaus Tracker
  2. Open Extensions → Apps Script
  3. Delete any existing code, paste the script from the apps-script.gs file included in your download
  4. Click Deploy → New deployment → Web app
  5. Set Execute as: Me · Who has access: Anyone
  6. Click Deploy → copy the Web app URL
  7. Paste it below and click Save
